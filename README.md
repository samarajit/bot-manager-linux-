# bot-manager-linux-
it is a tool that will manage your bots. like if you have multiole discord bot you can add them in bot manager and then it will run them stop them and also show the logs it is a basic version but feel free to contribute and expand it , i created this so that the bots i am hosting on rpi i can easily access them i hope you found it helpfull ^^


how to use 

git clone https://github.com/samarajit/bot-manager-linux-.git

cd bot-manager-linux-

pip install -r reqirements.txt

python bot_manager.py

you can access it at :
your ip::5000
